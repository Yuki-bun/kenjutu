# Keyboard Shortcuts

`Mod` = `Cmd` on macOS, `Ctrl` on Windows/Linux.

## Global

| Key     | Action               |
| ------- | -------------------- |
| `Mod+P` | Open command palette |

## Navigation

| Key                   | Action                           |
| --------------------- | -------------------------------- |
| `1`                   | Focus commit graph / commit list |
| `2`                   | Focus file tree                  |
| `3`                   | Focus diff view                  |
| `4`                   | Toggle comments sidebar          |
| `Mod+B`               | Toggle left sidebar              |
| `J` / `K`             | Focus next/previous item         |
| `Shift+J` / `Shift+K` | Scroll down/up                   |

## Commit Graph

| Key | Action                  |
| --- | ----------------------- |
| `C` | Copy change ID          |
| `D` | Edit commit description |

## File Tree

| Key      | Action            |
| -------- | ----------------- |
| `S`      | Focus file filter |
| `Escape` | Clear filter      |

## File Diff

| Key     | Action                          |
| ------- | ------------------------------- |
| `Space` | Toggle file reviewed/unreviewed |
| `Enter` | Enter line mode                 |
| `O`     | Toggle file expansion           |
| `C`     | Copy file path                  |
| `T`     | Toggle unified/split view       |

## Line Mode

Enter line mode with `Enter` on a file diff, exit with `Escape`.

| Key                 | Action                                   |
| ------------------- | ---------------------------------------- |
| `J` / `K`           | Move cursor down/up                      |
| `GG`                | Jump to top                              |
| `Shift+G`           | Jump to bottom                           |
| `Ctrl+D` / `Ctrl+U` | Half-page down/up                        |
| `N` / `Shift+N`     | Next/previous hunk                       |
| `V`                 | Toggle selection                         |
| `Space`             | Mark region as reviewed                  |
| `C`                 | Comment on selection                     |
| `Tab`               | Switch between Remaining/Reviewed panels |
| `Escape`            | Exit line mode                           |
